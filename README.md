# SuperX - Social Media Platform

## 🌟 Features

- 🔐 Authentication (Google & Email)
- 📝 Real-time Posts & Comments
- 👥 Friend System

- 🔔 Notifications

- [SweetAlert2](https://sweetalert2.github.io/)
